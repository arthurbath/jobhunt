# Job Search Helper

CLI/agent that researches target companies and PM-friendly roles, then syncs results into Airtable per `docs/AGENTS.md`.

## Getting Started
1. Install dependencies (to be defined once implementation stack is chosen).
2. Copy `.env.example` to `.env` and fill Airtable token/base ID.
3. Run the CLI (e.g., `jobhunt add-company "Example Co"`).

See `docs/AGENTS.md` for schema + workflow requirements.
